# Анализ пролонгации клиентских договоров

## Описание проекта

Цель проекта — оценить эффективность аккаунт-менеджеров по пролонгации клиентских договоров за 2023 год.

## Что было сделано

- Очистка финансовых данных
- Обработка пропусков и текстовых значений
- Агрегация дублей по id проекта
- Объединение двух источников данных
- Расчет коэффициентов пролонгации
- Формирование Excel-отчета для руководителя

## Используемые инструменты

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Excel

## Результаты

- Коэффициент пролонгации в первый месяц по отделу: 54,1%
- Коэффициент пролонгации во второй месяц по отделу: 7,4%

## Файлы

- `prolongation_analysis.ipynb` — код проекта
- `prolongation_report_2023.xlsx` — итоговый Excel-отчет
